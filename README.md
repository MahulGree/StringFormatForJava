# String format for Java

Примери за форматиране на стрингове в Java.

Илюстрира действието на метода String.format и употребата на класа StringBuilder

String.format - статичен метод на класа String, с богати възможности за форматиране и подравняване на стрингове, определяне на точността на изобразяване на числа с плаваща запетая, форматиране на дати, прибавяне на водещи нули и т.н.

StringBuilder - клас, който следва да се използва винаги, когато ще се форматира стринг, съставен от повече от един ред.
